# PWA Іконки

Ця директорія містить іконки для Progressive Web App.

## Необхідні іконки

Для повної підтримки PWA потрібні наступні іконки:

- `icon-192x192.png` - 192x192 пікселів (мінімальний розмір для Android)
- `icon-512x512.png` - 512x512 пікселів (рекомендований розмір)
- `apple-touch-icon.png` - 180x180 пікселів (для iOS)

## Генерація іконок

Запустіть скрипт генерації:

```bash
python web_admin/generate_icons.py
```

Або створіть іконки вручну, використовуючи будь-який графічний редактор.

## Вимоги до іконок

- Формат: PNG
- Прозорість: не обов'язкова, але бажана
- Колір: відповідає темі додатку (#0d6efd)
- Стиль: maskable (іконка може бути обрізана на різних пристроях)
